"Que: What is the difference between automatic and dynamic arrays?";"An automatic array is created on the stack with a size fixed at compile-time, while a dynamic array is allocated on the heap at runtime with a variable size using 'malloc' and must be freed manually. Simple Explanation: Автоматический массив – это как блокнот, который тебе выдают для работы, и его забирают, когда ты уходишь (размер известен заранее). Динамический – это как аренда склада; ты сам выбираешь размер и сам должен не забыть его освободить ('free'), когда он больше не нужен."

"Que: While using scanf, why is it important to use space before %?";"A space before a format specifier like '%c' in 'scanf' tells the function to skip any leading whitespace characters, such as spaces or newlines, that might be left in the input buffer from a previous entry. Simple Explanation: Это как сказать 'scanf': 'проигнорируй все случайные пробелы или нажатия Enter, которые остались от прошлого ввода, и только потом читай нужный символ'. Без этого, scanf может случайно прочитать символ новой строки от предыдущего нажатия Enter вместо символа, который вводит пользователь."
"Def: What is a null-terminator?";"The null-terminator is a single character, written as '\0', that is placed at the end of a string in C to mark its termination. Simple Explanation: Это специальный невидимый 'стоп-знак' в конце каждой строки. Функции вроде printf бегут по буквам и останавливаются, только когда видят этот знак. Без него они бы не знали, где заканчивается строка, и читали бы дальше мусор из памяти."
